What is CryptoWatch?

CryptoWatch is a real-time cryptocurrency tracking web application. The goal of this project is to help users easily monitor the prices of various cryptocurrencies. By using trusted data sources like CoinGecko and CryptoCompare, CryptoWatch allows you to search for cryptocurrencies, view their latest prices, and generate custom reports to track your selected coins over time.

The app allows users to:

  • Search for cryptocurrencies by symbol.
  • Track up to 5 coins in real-time with live pricing data.
  • View interactive reports with detailed price history for selected cryptocurrencies.
  • Customize your experience by adding or removing coins with simple toggle switches.
Technologies Used in CryptoWatch

CryptoWatch is built using modern web technologies to provide a seamless, responsive experience. Key technologies include:

  • HTML & CSS: Structuring and styling the app for responsiveness and accessibility.
  • JavaScript (ES6+): For dynamic functionality like fetching live data and rendering charts.
  • Bootstrap: For mobile-first, responsive design that works across devices.
  • Chart.js: For visualizing cryptocurrency price trends and reports.
  • APIs (CoinGecko, CryptoCompare): Fetching real-time cryptocurrency data for accurate and up-to-date information.
